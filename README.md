📦 JSON Server API
Fake REST API được tạo bằng JSON Server phục vụ cho dự án

🚀 Deploy URL (Production)
https://api-antoree.onrender.com

▶️ Cách chạy ở local (Development)

1. Cài đặt dependencies:

```bash
npm install
```

2. Chạy JSON Server:

```bash
npm run dev
```

✅ Server sẽ chạy tại http://localhost:3000
